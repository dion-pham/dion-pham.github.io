Projects

aroundTwo GitHub

Full stack clone of Round Two Store. aroundTwo allows users to browse, leave reviews and simulate the purchase of a variety of streetwear and vintage inspired clothing items.

Utopia GitHub

Utopia is a full stack web application inspired by OkCupid. Users can create an account and fill out a questionnaire to see who they match with. From there, a user can reach out to a match with live-messaging through the implementation of Socket.io.

PhamBnB GitHub

PhamBnb is a full stack web application inspired by AirBnb. It can be used to browse, book and host new spots around the world, leaving reviews on spots worth highlighting.
